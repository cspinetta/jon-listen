Jon Listen
=================================
An UDP listener to forward data to a file.

Written in [Rust] language.

## Dependencies

- Rust nightly:

```bash
rustup install nightly
rustup default nightly

```

## Start server

Execute from the terminal:

```bash
cargo run
```

## Run benchmarks

Execute from the terminal:

```bash
cargo bench
```
